# Spring Boot Todo API

API REST simples para gerenciamento de tarefas desenvolvida com **Java** e **Spring Boot**.

A aplicação permite criar e listar tarefas utilizando operações CRUD e utiliza banco de dados em memória **H2** para armazenamento. Também possui uma interface simples em **HTML** e **CSS** para visualização das tarefas no navegador.

## 🚀 Tecnologias utilizadas

- Java
- Spring Boot
- Maven
- H2 Database
- HTML
- CSS
- REST API

## 📌 Funcionalidades

- Criar tarefas
- Listar tarefas
- Armazenamento em banco de dados
- Interface web simples para visualizar tarefas
